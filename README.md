# safetyclock
基于 STM32 的多功能安全时钟与环境监测系统
本项目设计了一款兼具时间管理与室内消防预警功能的嵌入式系统 。系统核心采用主频为 72MHz 的 ARM Cortex-M3 架构 STM32F103C8T6 微控制器 。整体系统集成了 OLED 显示、时钟校准、倒计时管理及环境气体监测等多任务模块 。

技术实现:
安全预警机制开发： 引入 MQ-2 气体传感器实时监测室内烟雾及甲烷等可燃气体浓度 。利用 STM32 的 GPIO 及 ADC 接口采集传感器模拟电压信号 。结合传感器 200-10000ppm（精度 ±10%）的测量范围设定安全阈值 。当浓度超标时，通过 PA0 引脚控制 PNP 三极管，驱动蜂鸣器触发非阻塞式的高频声光报警 。

底层外设与通信驱动编写： 基于 C 语言进行裸机开发 。通过 GPIO 模拟 I2C 通信时序，独立编写并封装了 OLED12832 显示屏与 TM1637 四位数码管的底层驱动程序，实现了多格式数据的稳定异步刷新 。

低功耗与智能唤醒逻辑设计： 针对卧室场景优化系统功耗，设计了基于时间段的智能休眠与唤醒调度机制 。白天接入 SR501 人体红外传感器监测人体活动 。夜间切换至 KY-037 声音传感器识别环境异常噪声 。触发唤醒后，显示屏及数码管工作 1 分钟后自动清屏复位，有效降低了系统的待机功耗 。

系统定时器调度与测试： 配置并管理 TIM2、TIM3 和 TIM4 三个独立硬件定时器 。分别用于实现高精度的软件时钟计数、显示界面的时长保持，以及蜂鸣器状态机的中断轮询 。通过中断优先级的合理分配，保障了在多传感器并发检测下系统的实时性与稳定性 。
